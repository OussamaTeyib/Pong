## Pong Game
This is an implementation of the classic Pong game, built using [raylib](https://github.com/raysan5/raylib).

### Usage
1. Clone the repository.
2. Build and run the excutable:
   ```
   cmake -B Build
   cmake --build Build
   ```
3. Enjoy the game!
